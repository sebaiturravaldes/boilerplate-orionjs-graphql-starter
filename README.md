# Orionjs Graphql Starter

## Requirements

Node >= 14 (LTS)

## Environment Variables

copy `.env.example` file to `.env` with command:

```
cp .env.example .env
```

## Install

```
yarn install
```

## Run develop

```cli
orion start
```

Open your browser and navigate to [http://localhost:3000/graphiql](http://localhost:3000/graphiql)

## Build

```cli
yarn build
```
